# Dog Training Methods Analysis Project

A data analysis project created in Google Sheets.

## Project Overview
This project analyzes the effectiveness of different dog training strategies (Positive Reinforcement, Negative Reinforcement, Positive Punishment, and Negative Punishment) using Google Sheets. The analysis includes:

- Collecting simulated dog training data  
- Calculating percent of successful outcomes per method  
- Calculating average time to learn per method  
- Calculating success scores with numeric conversion (1 = Success, 0.5 = Partial, 0 = Failure)  
- Creating charts to visualize performance metrics for each method  

---

## Analysis Tasks

| Task | Samples | Data Source | Notes |
|------|--------|------------|------|
| Count by Method | 30 simulated dog training scenarios | Raw_Data tab | Number of dogs trained using each method |
| Percent Success | Calculated from success outcomes | Raw_Data tab | Success rate per method (partial and failure excluded) |
| Average Time | Calculated from days to learn | Raw_Data tab | How quickly dogs learned each method |
| Success Score | Numeric conversion (1, 0.5, 0) | Raw_Data tab | Combines partial and full success |
| Charts | Visual representation of metrics | Google Sheets (derived from tables) | Column charts with consistent formatting |
